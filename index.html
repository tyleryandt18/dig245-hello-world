learn-computing Summary:

I chose to learn about APIs because I want to work with data when I graduate. APIs are used everywhere; they connect data and/or other functionality to the software that uses them. If I use an API, I can specify my requests, similarly to a search engine. Most APIs are free, but some require authentication. The URL of an API can be broken down into three parts; the 'https', which provides security over computer networks, the API's host, and then lastly the information that follows the '/' after the host. This is essentially the search engine's criteria. I practiced by using the Spotify API. I pulled my top 5 songs from the past 30 days (all were from the new Travis Scott album, Utopia), and got new recommendations.
